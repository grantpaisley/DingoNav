Config

The complete setup in one JSON file — settings, tuning overrides, zoom presets, screen layouts, controller keys. Export downloads the file and copies it to the clipboard; bench-tune on a desktop browser, export there, import here.
Demo replays the selected track — use it to hear the alerts before a real ride.

VargPilot server

VargPilot holds the bike's Bluetooth and re-serves telemetry from its built-in webserver. Running VP on this phone → http://127.0.0.1:8080; on another device use the URL VP displays (same Wi-Fi).

Direct BLE (no VargPilot)

The app talks to the VCU itself — it then owns the bike's single BLE slot; tap the 🔋 pill to hand it to the Stark app and back.

Controller buttons

Works with any Bluetooth remote that types keys (e.g. a DMD2 handlebar controller). Tap an action, then press the button on your remote. Esc while capturing clears a custom key.
Drag buttons and readouts anywhere; drag the cyan corner dot on a readout to resize it; drop a button on the bottom tray to dock it in this menu. Saved separately for portrait and landscape.

Tuning

Live tuning knobs — changes apply immediately and persist on this device. ↺ resets a row to default.
